# Sorting Visualizer

A clean and interactive Data Structures and Algorithms mini project that helps users understand how sorting algorithms work through live animation.

## Live Project

[Open Website](https://charanvenkatasaikrishna.github.io/sorting-visualizer-website/)

## Overview

This project visualizes sorting algorithms by representing array elements as vertical bars. As the algorithm runs, the bars update to show comparisons, swaps, and sorted positions step by step.

It is designed to make DSA concepts easier to learn, present, and explain.

## Features

- Generate random arrays
- Enter custom array values
- Visualize Bubble Sort
- Visualize Selection Sort
- Visualize Insertion Sort
- Adjust sorting speed
- View comparisons and swaps
- See time and space complexity
- Reset the array to its original state

## DSA Concepts Used

## Algorithms Included

### Bubble Sort

Bubble Sort repeatedly compares adjacent elements and swaps them if they are in the wrong order.

### Selection Sort

Selection Sort repeatedly selects the smallest remaining element and places it in the correct position.

### Insertion Sort

Insertion Sort builds the sorted portion one element at a time by inserting each value into its correct place.
